# Geochemical Multivariate Analysis

This repository contains a complete workflow for multivariate analysis of geochemical data,
including PCA, Factor Analysis, correlation analysis and hierarchical clustering.

## Project structure

- src/geochemical_analysis.py : main analysis script
- data/example_dataset.xlsx : example input dataset
- results/ : output files generated by the script

## Requirements

Install the required libraries using:

pip install -r requirements.txt

## How to run

Run the script with:

python src/geochemical_analysis.py

## Notes

- Option 1 uses median imputation (SPSS-like behavior)
- Option 2 uses only complete samples (conservative approach)
